## ğŸ“ Customization Guide

### Update Contact Information

In `index.html`, line 263:
```html

```
Change `tu@email.com` to your actual email.

In `styles.css`, update the color variables:
```css

--construction-yellow: #f59e0b;
```

### Add Real Images

Create an `images/` folder and update image paths in `index.html`
